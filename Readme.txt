=========================================================================
Realizar un Side Project que permita ver las banderas de los países de habla hispana de tal modo que se pueda ver la información general de cada uno de ellos.
para la fase 1 trabajaremos con datos locales, en la fase 2 se estará implementando una api para este mismo servicio.

Funcionalidades:

1. Al iniciar la aplicación se cargará una sección con cada una de las banderas y su respectivo nombre.
2. Al pasar el mouse por encima de alguna de las tarjetas, esta mostrará la información general.
3. Se podrá filtrar por nombre del país o su capital
4. Se tendrá la opción de ordenar los países alfabeticamente.

Tareas:

1. Analizar el mockup de la aplicación en desktop(Web)
2. Revisar la semántica de la app.
3. escirbir los estilos.
4. Definir las funciones y los scripts.
5. desplegar
